Tingkatkan Karirmu dengan CV-Match AI

Unggah CV-mu dan biarkan teknologi Natural Language Processing (NLP) kami mencarikan pekerjaan yang paling sesuai dengan keahlianmu secara semantik.

Pilih atau Tarik File CV ke Sini
Format yang didukung: .PDF (Maks 5MB)

⚙️ Alur Proses AI & Pipeline Sains Data

Penjelasan teknis dan panduan optimalisasi CV Anda untuk melewati setiap tahapan evaluasi algoritma NLP kami

Tahap 1
Data Gathering

AI membaca file CV PDF secara dinamis dan memuat kamus dataset karir industri (73 peran pekerjaan aktif).

Instruksi CV: Unggah file PDF asli (berbasis teks). Hindari hasil scan foto agar teks terbaca sempurna oleh library PyPDF2.
Tahap 2
Text Preprocessing

Teks dibersihkan dari simbol asing, angka, spasi berlebih, dan diseragamkan ke huruf kecil (Case Folding).

Instruksi CV: Gunakan tata letak sederhana (1 kolom). Jangan taruh teks penting di dalam kotak grafis agar tidak pecah saat diekstrak.
Tahap 3
Feature Extraction

Mengekstrak dan menghitung bobot kata kunci terpenting di CV Anda menggunakan pembobotan statistik TF-IDF n-gram.

Instruksi CV: Tulis keahlian teknis secara eksplisit (seperti 'Python', 'SQL', 'Figma') daripada sekadar grafik persentase abstrak.
Tahap 4
Semantic LSA

Menghitung kesamaan makna kata (sinonim) lewat model LSA (SVD) untuk menangkap konteks keahlian secara matematis.

Instruksi CV: Jabarkan pencapaian proyek Anda dengan deskripsi mendalam agar model LSA dapat menangkap konteks kalimat Anda secara logis.
Tahap 5
Expert System Scorer

Sistem Pakar menguji kehadiran Action Verbs, metrik pencapaian (angka/%), serta profiling Soft Skills berbasis kamus kepribadian.

Instruksi CV: Cantumkan angka/data kuantitatif konkret (misal: 'menaikkan penjualan 25%') dan pakai kata kerja aktif kuat.

AI sedang membaca dan menganalisis CV-mu...

Memproses: Preprocessing → TF-IDF Fitur → Pemodelan Semantik LSA → Expert Scoring
